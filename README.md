# StudyBuddy
StudyBuddy is a frontend-based web application that helps students manage their academic activities efficiently. It includes features like task management, study timer, attendance tracking, and note-taking, all stored locally in the browser. The system is designed to improve organization, time management, and overall productivity.

## Features

- Dashboard for quick overview
- Task management with add, edit, delete, and completion tracking
- Study timer for focused sessions
- Attendance tracker
- Notes section for class and revision notes
- Smart suggestions for better productivity
- Support for bunkable classes and planning
- Clean and responsive user interface

## Tech Stack

- HTML
- CSS
- JavaScript

## Future Improvements

- Cloud sync
- User login
- Mobile app version
- Notifications and reminders
- Better analytics for study habits
- Image upload support in Notes
- Downloadable notes in pdf format
